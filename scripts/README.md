# scripts/

This directory contains the EDM → mocho transform pipeline and the lookup tables it consumes.

## transform/

A Python package that converts DDB EDM JSON records (exported as SQLite by the ingest pipeline) to N-Quads partitioned into named graphs.

See [`transform/README.md`](transform/README.md) for usage, inputs/outputs, and design decisions.

Architecture Decision Records:
- [`../docs/adr/transform-adr.md`](../docs/adr/transform-adr.md) — class dispatch and WEMI alignment decisions
- [`../docs/adr/transform-props-mapping-adr.md`](../docs/adr/transform-props-mapping-adr.md) — property mapping decisions
- [`../docs/adr/transform-script-adr.md`](../docs/adr/transform-script-adr.md) — implementation decisions

## config/

Lookup tables consumed by `transform/` at runtime. Generated by the corpus analysis scripts in [`../goethe-faust/scripts/`](../goethe-faust/scripts/).

| File | Purpose |
|---|---|
| `lookup_dctype_to_class.csv` | dc:type string → mocho class dispatch table |
| `lookup_htype_doco_rico.csv/json` | htype code → mocho/DoCO/RiC-O class mapping |
| `lookup_class_prop_alignment.csv` | (target_class, edm_prop) → target_prop alignment |
| `image_type2class.json` | IMAGE mediatype dc:type → class mapping |
| `audio_type2class.json` | AUDIO mediatype dc:type → class mapping |
| `video_type2class.json` | VIDEO mediatype dc:type → class mapping |
| `lido_event_types.csv` | LIDO event type → RDA predicate mapping |
| `loc-iso-639-2-utf-8.txt` | ISO 639-2 language code lookup |

## Transform → N-Quads → QLever workflow

```
SQLite (per sector, from ingest pipeline)
        │
        ▼
python -m transform \
  --input  <sector.sqlite> \
  --config scripts/config/ \
  --output <sector.nq>
        │
        ▼
N-Quads (named graphs: graph/ddbedm, graph/mocho, graph/prov)
        │
        ▼
qlever index build
  qlever start --index gemea --nq-file gemea.nq
        │
        ▼
QLever SPARQL endpoint (default: http://localhost:7030)
SHMARQL browser   (default: http://localhost:7032)
```

Run `python -m transform --help` for full CLI options.
